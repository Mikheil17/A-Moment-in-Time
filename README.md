# A Moment in Time
A short glimpse into a dreamscape of peace and colorful lands, where all your worries are whisked away by serene melodies and the sounds of waves crashing against your boat.

## What It Is
A short Unity experience designed to immerse the user in a moment of calm, surrounded by breathtaking scenery.

## Tech
Unity | C#

## Context
Created as part of a VXR class as my first Unity learning project.
